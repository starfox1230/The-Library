Speed Streak is an add-on I made because I am chronically slow with Anki. The point is to turn review into a game of speed: answer quickly or admit you do not actually know the card yet. It is meant to help you stop getting trapped by a handful of cards and protect the main review session so your due cards get done today.

What matters most to me is that it makes time feel real. There is a timer on both sides of the card, a streak you do not want to lose, a score that rewards staying on pace, and visual feedback that makes momentum feel satisfying. If you use a haptic-enabled controller, good answers feel like landing hits and timeouts feel like you have done something wrong in a way that is weirdly motivating.

This is not really about mindless rushing. It is about making hesitation visible, making time cost something, and helping you quickly sort cards into three buckets: cards you know, cards you do not know, and cards that are badly timed for a speed-focused session.

What it does

  • Runs a live question timer and answer timer so every card has a visible cost
  • Builds a streak, score, and growing orbit so momentum feels rewarding and losing it feels meaningful
  • Adds controller haptics on compatible setups so reveals, answers, skips, and time pressure feel physical instead of abstract
  • Lets you flag certain cards as Time Drain so they are called out immediately and can be buried before they derail the whole session
  • Lets you flag cards as Review Later so you can keep moving now and come back when you actually have time
  • Includes a Review Later Manager that can filter those cards, copy them as text, open them in the Browser, or build a filtered deck from them

The philosophy

A speed session is for getting through due cards efficiently. If you know a card, answer it fast. If you cannot answer it fast, that is useful information. If a card is inherently too long, poorly designed, or just wrong for the moment, the session should not be sacrificed to it. Speed Streak is built to help you notice that quickly and keep moving.

Time Drain

Time Drain is for cards that repeatedly eat too much time. They are not necessarily bad enough to suspend or delete, but they are bad for a speed session. When one appears, Speed Streak warns you clearly so you can bury it, preserve momentum, and deal with it later on purpose instead of accidentally spending half your review block on it.

Review Later

Review Later is for cards you got through but still want to revisit more carefully. It gives you a way to protect today's pace without pretending the card no longer matters.

Review Later Manager

The Review Later Manager is there for the second pass.

  • View all cards currently marked Review Later
  • Filter them by when they were added, such as Today, Yesterday, Past X days, or All
  • Copy the visible cards as plain field text
  • Open the visible set in Anki Browser
  • Create a dated filtered deck from the visible set

Other useful details

  • Pause and resume with P
  • Question and answer timers can be customized in Settings
  • You can keep the top card timer visible during review
  • There are appearance options, saved color customization, and a vibration-only mode
  • Historical stats tracking is included and still being expanded

Notes

  • Desktop Anki add-on only
  • Not for AnkiWeb or mobile
  • Windows with an XInput-compatible controller is still the most reliable setup for haptics
  • Non-Windows platforms may still work visually, but controller vibration depends on the platform, browser/gamepad support, and controller setup
  • If you are on a slower machine, you can tone down or disable the heavier visual elements

High-yield recent changes

  • Improved Mac compatibility for the core visual overlay
  • Added a compatibility window mode to play more nicely with add-ons like AMBOSS and AnkiHub
  • Undo now also rolls back your score and streak
  • Added lower-resource visual options for slower computers
