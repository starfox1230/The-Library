The companion only displays what the bridge publishes. If the visuals and Anki drift, hit F8 again on a question side to resync.
Phase rules
Question side uses the answer timer. Reveal with Space. After reveal, review side uses the rating timer. Rating keys should only score after reveal.
Scoring
Base: Again 0, Hard 2, Good 3, Easy 4.

Answer bonus = max(0, ceil((12 - answer seconds) / 2)) when the card is timed.
Review bonus = max(0, ceil((8 - review seconds) / 2)).

Every 10 on-time cards adds +10% combo.
Recommended flow
Start on an unrevealed question. Press F8 once to sync. The first synced card is free so the companion does not guess when that question began. After that, each newly arrived card starts its own answer timer immediately.
