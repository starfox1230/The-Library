Anti-Scattcr Grid: Since the breast is placed in compression and you are using lower
kVp (both of which intrinsically reduce scatter) a smaller grid ratio is used.
Mammo uses a 4-5 grid ratio (general x-ray uses a 6-16).
Dose with the Grid-The grid removes scattered photons-
which reduces some dose. However, the same technique
can't be used with a grid or you will underexpose. So, the
technique is turned up. Ultimately the dose is
INCREASED with a grid.
