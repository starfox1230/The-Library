🧪 Temporary Apps Library

A staging area for short-lived tools. Newest entries appear first by default.

Systematic Add Instructions (for humans + bots)

  1. Create a new app folder at: apps/temporary-apps/library/YYYY-MM-DD-short-slug/.
  2. Add the app page at index.html inside that folder.
  3. Add one new object at the TOP of TEMP_APPS in this file.
  4. Use ISO date strings (YYYY-MM-DD) for both createdAt and updatedAt.
  5. Keep newest apps first by sorting with updatedAt desc, then createdAt desc.
  6. Before committing, run python3 scripts/verify-temporary-apps-index.py from the repo root.
