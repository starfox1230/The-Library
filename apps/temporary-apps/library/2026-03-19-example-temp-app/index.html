Quick guidance

Plain version

What you likely need to update in ERAS

After registering with NRMP, add your NRMP ID to your MyERAS profile. That is the most clearly supported post-application ERAS step from the official guidance.

What counts for the match

Interviews and ERAS are not the same thing as NRMP registration. You still have to register in NRMP and later certify your rank list for it to count.

How to handle the project share-out

  • New clean GitHub account for your public-facing identity.
  • One discrepancy-project repo there.
  • One Anki add-on repo there.
  • Keep the live Render deployment where it already works.
  • Send people the live demo plus the clean repo.

Minimum viable handoff package

  • Live demo link.
  • Repo link.
  • Three to five example cases.
  • One sentence asking what kind of feedback you want.
