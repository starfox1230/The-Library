Legitimate Anki Add-on

Anki Pocket Knife

A single add-on that bundles your early-review shortcut with a set of missed-today tools.

  • Launcher: Ctrl+Shift+Q if Anki has not already claimed it
  • Early Review: keeps the Ctrl+Alt+F workflow when it is safe to do so
  • Missed Today: copy text, save a text export, open a local HTML viewer, or create a no-reschedule filtered deck
  • No-Image Today: choose one or more deck levels or use the deck gear menu to build a rescheduling filtered deck from today's surfaced question-side no-image cards
  • Suspended Browser View: open all suspended cards in the Browser with the most recently suspended cards first
  • Filtered Deck Cleanup: send only the non-new cards in a filtered deck back to their original decks while keeping their current schedule
  • Add Cards Picture Frame: default-on pink picture-frame button that converts the current cloze note into Visual_Card_Multitude by splitting cloze text, question text, images, and Extra into the right fields, and converts back to saCloze++ when pressed on a Visual_Card_Multitude note
  • Add Cards Auto Deck: top-bar toggle that can switch cloze notes between .New::Audio and .New::Visual based on images in the Text field, and optionally forces Visual_Card_Multitude into .New::Visual
  • Auto-Scroll: optional toggle that jumps back to the top of the card when you reveal the answer

Install on Windows

Run this from PowerShell to install the add-on into your Anki addons21 folder:

Set-Location "C:\Users\sterl\OneDrive\Documents\GitHub\The-Library\apps\anki-pocket-knife"
powershell -ExecutionPolicy Bypass -File .\install_to_anki.ps1

What is in this folder

menu.py Registers the launcher dialog, menu, shortcuts, and toggleable reviewer behaviors.
early_review.py Reuses and preserves the early-review filtered-deck workflow.
missed_today.py Handles today's misses, clipboard export, text export, HTML viewer generation, and filtered decks.
no_image_today.py Selects deck-level surfaced cards for today, filters out question-side image cards, and builds a new filtered deck.
suspended_browser.py Opens the Browser to all suspended cards and preserves a recent-suspension ordering so the newest suspensions rise to the top.
return_non_new.py Lets you choose a filtered deck and send only its current non-new cards back home without disturbing the new cards still in that deck.
visual_card_multitude.py Adds the pink picture-frame button to Add Cards and converts between your cloze note type and Visual_Card_Multitude in one click.
install_to_anki.ps1 Installs the add-on while preserving runtime files in user_files.

Build a package

This creates anki_pocket_knife.ankiaddon in the same folder:

Set-Location "C:\Users\sterl\OneDrive\Documents\GitHub\The-Library\apps\anki-pocket-knife"
powershell -ExecutionPolicy Bypass -File .\build_ankiaddon.ps1
