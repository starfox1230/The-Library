Legitimate Anki Add-on

Early Review Deck

This folder now contains a real Anki add-on with a proper __init__.py, manifest.json, and Windows installer/build scripts.

  • Shortcut: Ctrl+Alt+F
  • Prompts for a card count, defaulting to 50
  • Finds review cards due tomorrow and picks the longest intervals first
  • Builds or refreshes ............... Early Preview Deck for YYYY-MM-DD

Install on Windows

Run this from PowerShell to install the add-on into your Anki addons21 folder:

Set-Location "C:\Users\sterl\OneDrive\Documents\GitHub\The-Library\apps\anki-addon-early-review"
powershell -ExecutionPolicy Bypass -File .\install_to_anki.ps1

What is in this folder

__init__.py The actual add-on entrypoint Anki loads.
manifest.json Declares the add-on name and package folder.
install_to_anki.ps1 Copies the add-on into Anki for local use.
build_ankiaddon.ps1 Creates a clean .ankiaddon package for sharing.

Build a package

This creates early_review_deck.ankiaddon in the same folder:

Set-Location "C:\Users\sterl\OneDrive\Documents\GitHub\The-Library\apps\anki-addon-early-review"
powershell -ExecutionPolicy Bypass -File .\build_ankiaddon.ps1
